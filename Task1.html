what is Browser?

A browser is a computer program you use to visit websites on the internet

Example:Google Chrome, Safari, Firefox, and Microsoft Edge

what is Internet?

The internet is a big network that connects millions of computers and devices around the world

Example:Watching videos on YouTube

what is Domain?

A domain is the name of a website that you type in the address bar to visit it

Example:In www.google.com, the domain is google.com

what is IP Address?

An IP address is a unique number given to each device connected to the internet

Example:An IP address might look like this: 192.168.1.1

what is URL?

A URL (Uniform Resource Locator) is the full web address you type to visit a specific page on a website

Example:https://www.youtube.com/watch?v=abc123

what is HTML?

HTML (HyperText Markup Language) is the basic language used to create web pages

what is XHTML?

XHTML stands for Extensible Hypertext Markup Language. It's a hybrid language that incorporates HTML and XML. XHTML is nearly identical to HTML, although it is more stringent
